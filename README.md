# React Architecture Lab

Shell inicial em React + TypeScript + Tailwind para documentar e demonstrar arquitetura front-end escalavel.

## Rodar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Ideia do projeto

Este shell foi criado para virar um portfolio tecnico/documentacao viva sobre:

- Arquitetura front-end
- React + TypeScript
- Feed estilo Twitter/Instagram
- Performance
- Dados/API
- Tempo real
- Design System
- Qualidade
- Infra/deploy
- Microfrontends futuros
